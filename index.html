My Process

I try to follow the Design Thinking methodology for every project I take part in. Using this methodology allows me to deliver human centered solutions to the problem at hand.

01

Research

Before jumping into design I always make sure to empathize with the user needs. This step includes user interviews, surveys, and benchmark.

02

Design

Once framed the real user problem I start to create a solution for it. This step can range from low fidelity wireframes to high fidelity interactive prototypes.

03

Test

I'm always happy to receive user feedback! This phase allows me to validate my solution and to find and solve any possible problem before its implemented.
